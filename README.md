# goodpicker

## Install

### 1. Backend

- python3
- pipenv  
  `sudo pip3 install pipenv`
- virtualenv  
  `sudo pip3 install virtualenv`
- pew  
  `sudo pip3 install pew`
```
pipenv shell
pipenv install
```
### 2. Frontend

```
cd frontend
npm i
```

## Start

### 1. Backend

```
pipenv shell
cd backend
python manage.py runserver
```

### 2. Frontend

```
cd frontend
npm start
```
